JavaScript Encapsulation

It is process of binding the data with the function acting on that data. It allows us to control the data and validate it.

Points to remember..!
  • Use var keyword to make data members private.
  • Use setter method to set the data and getter method to get the data.

Encapsulation allows us to handle an object using the following properties.

Read/write - Here, we use setter methods to write the data and getter methods read that data.

Read only - In this case, we use getter methods only.

write only - In this case, we use setter methods only.
